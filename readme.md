# MyApp

Repositori ini berisi kumpulan latihan dan contoh kode C# untuk belajar dasar-dasar bahasa pemrograman C# menggunakan .NET 10.0.

## Tentang

Proyek ini dibuat sebagai catatan dan latihan dari kursus video yang tersedia di YouTube:

- https://www.youtube.com/watch?v=YrtFtdTTfv0

Proyek utama adalah aplikasi konsol sederhana dengan contoh `FizzBuzz` di `Program.cs`, sekaligus mencakup banyak topik dasar C# dalam berkas README terpisah.

## Cara Menjalankan

1. Buka terminal pada folder `MyApp`.
2. Jalankan perintah berikut:

```bash
dotnet run
```

3. Ikuti instruksi pada konsol. Program saat ini akan meminta input angka dan menampilkan hasil permainan `FizzBuzz`.

## Tujuan

Repositori ini ditujukan untuk:

- Memahami konsep dasar C# dan .NET 10.0
- Membiasakan diri dengan struktur proyek `.csproj` dan solusi `.sln`
- Mempelajari input/output konsol, operator, kondisi, dan perulangan
- Mengerjakan latihan praktis seperti tabel perkalian dan FizzBuzz

## Catatan

Setiap `readme*.md` berisi ringkasan materi atau latihan dari topik C# tertentu. Gunakan file-file ini sebagai panduan belajar dan referensi saat mengembangkan kode di `Program.cs` atau membuat latihan tambahan.
