# MyApp

Repositori ini berisi kumpulan latihan dan contoh kode C# untuk belajar dasar-dasar bahasa pemrograman C# menggunakan .NET 10.0.

## Tentang

Proyek ini dibuat sebagai catatan dan latihan dari kursus video yang tersedia di YouTube:

- https://www.youtube.com/watch?v=YrtFtdTTfv0

Proyek utama adalah aplikasi konsol sederhana dengan contoh `FizzBuzz` di `Program.cs`, sekaligus mencakup banyak topik dasar C# dalam berkas README terpisah.

## Struktur Proyek

- `MyApp.sln` - solusi .NET untuk proyek C#.
- `MyApp.csproj` - proyek aplikasi konsol .NET 10.0.
- `Program.cs` - aplikasi FizzBuzz sederhana.
- `readme*.md` - berkas tutorial dan latihan per topik:
  - `readme1.md` - topik dasar awal.
  - `readme2(convertingString).md` - konversi string dan tipe data.
  - `readme3(string).md` - manipulasi string.
  - `readme4(operators).md` - operator dasar.
  - `readme5(modulus).md` - operator modulus.
  - `readme6(varKeyword).md` - penggunaan `var`.
  - `readme7(constKeyword).md` - penggunaan `const`.
  - `readme8(excercise).md` - latihan dasar.
  - `readme9(excercise).md` - latihan tambahan.
  - `readme10(inputOutput).md` - input/output konsol.
  - `readme11(ifStatements).md` - percabangan `if`.
  - `readme12(switchCase).md` - `switch` dan `case`.
  - `readme13(for loops).md` - perulangan `for`.
  - `readme14(whileLoops).md` - perulangan `while`.
  - `readme15(conditionalOperator).md` - operator kondisional.
  - `readme16(numericFormating).md` - format angka.
  - `readme17(tryParseFunction).md` - penggunaan `TryParse`.
  - `readme18(excerciseTimesTable).md` - latihan tabel perkalian.
  - `readme19(excerciseFizzBuzz).md` - latihan FizzBuzz.
  - `readme(boolean).md` - contoh penggunaan tipe boolean.

## Cara Menjalankan

1. Buka terminal pada folder `MyApp`.
2. Jalankan perintah berikut:

```bash
dotnet run
```

3. Ikuti instruksi pada konsol. Program saat ini akan meminta input angka dan menampilkan hasil permainan `FizzBuzz`.

## Tujuan

Repositori ini ditujukan untuk:

- Memahami konsep dasar C# dan .NET 10.0
- Membiasakan diri dengan struktur proyek `.csproj` dan solusi `.sln`
- Mempelajari input/output konsol, operator, kondisi, dan perulangan
- Mengerjakan latihan praktis seperti tabel perkalian dan FizzBuzz

## Catatan

Setiap `readme*.md` berisi ringkasan materi atau latihan dari topik C# tertentu. Gunakan file-file ini sebagai panduan belajar dan referensi saat mengembangkan kode di `Program.cs` atau membuat latihan tambahan.
